Deep Dive: matplotlib¶

  • Visualization library for Python.
  • Produces 2D and 3D plots and graphs.
  • Highly customizable.
  • The most common submodule is pyplot, normally imported as plt.
In [1]:
import matplotlib.pyplot as plt
In [ ]:
x = [10, 20, 30, 40, 50]
y = [1, 2, 3, 4, 5]

  • Offers DataFrames (tabular data structures).

Indices¶

  • Indices are unique identifiers for each row.
  • By default, the range is [0, dataframe's length - 1].
  • As long as their values are unique, we can use any values we want as an index.

Multiple filters¶

  • Use & for AND operations.
  • Use | for OR operations.
  • Parentheses are crucial for multiple conditions.

Merging and Joining¶

  • Combine DataFrames based on common columns.
  • Types: inner, outer, left, right.
In [122]:
df1 = pd.DataFrame({'key': ['A', 'B', 'C'], 'value1': [1, 2, 3]})
df2 = pd.DataFrame({'key': ['A', 'B', 'D'], 'value2': [4, 5, 6]})

# Inner merge (intersection)
